# Rex大叔的168台股投資教室 - AI智慧台股熱門交易系統

## 系統簡介

這是一個多功能的台股熱門交易資訊查詢系統，整合了可轉債(CB)、上市櫃股票、法人籌碼等多維度數據分析。

## 功能特色

### 左側面板 - 股票查詢功能
- 輸入股票代號或名稱進行查詢
- 以卡片形式展示股票完整資訊
- 包含價格、漲跌幅、成交量、法人籌碼等關鍵數據

### 右側面板 - 熱門交易排行
- **可轉債CB**：成交值、週轉率、漲跌幅、計量強弱勢、200日新高低、法人買超
- **上市櫃股票**：成交值、週轉率、當沖率、計量強弱勢排行
- **法人籌碼**：外本比/投本比 買超賣超排行
- **供應鏈**：NVIDIA概念股、台積電供應鏈

## 使用方式

### 本地使用
1. 將所有檔案（index.html 及所有 CSV 檔案）放在同一個資料夾
2. 使用瀏覽器開啟 index.html 檔案
3. 由於瀏覽器安全限制，建議使用本地伺服器：
   - Python 3: `python -m http.server 8000`
   - Node.js: `npx http-server`
   然後在瀏覽器開啟 http://localhost:8000

### GitHub Pages 部署
1. 將所有檔案上傳到 GitHub 倉庫
2. 開啟倉庫的 Settings → Pages
3. 選擇 main branch 作為來源
4. 等待部署完成後即可使用

## 操作說明

1. **查詢股票**：在左側搜尋框輸入股票代號（如 2330）或名稱，按查詢或 Enter
2. **切換報表**：點擊右上方的類別按鈕，選擇想查看的報表
3. **排序**：點擊表頭欄位可進行升降序排列
4. **查看詳情**：點擊表格中的股票代號，左側會顯示該股票的詳細資訊卡片
